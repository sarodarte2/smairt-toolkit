Core documentation for the SMAIRT framework.

- `12_STEPS.md` - The 12 steps of the SMAIRT workflow

## The Core Idea

"The template is the structure. We provide code for logging, for the structure of the naming conventions, and for the prompts—especially with that structure of how we're going to structure things. We're going to structure them exactly like the scientific method and it's going to be an iterative process where we're going to record all of these things."
