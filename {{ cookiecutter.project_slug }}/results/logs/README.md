# Output Logs

Store all script output logs here.

## Naming Convention

Log files should be named to match their scripts:
script_01_initial_test_output.log
script_02_add_noise_output.log



"It should provide output on the command line but really it should provide an output log file that is named the same thing as the script. That way you have all the log files in one place and you have the scripts which are closely associated with that."

## Purpose

These logs serve as:
1. A record for you to reference later
2. A record for AI—feed these back to recreate context
3. Part of the breadcrumb trail through your experiments
