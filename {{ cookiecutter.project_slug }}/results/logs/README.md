# Output Logs

Store all script output logs here.

## Naming Convention

Log files should be named to match their scripts:
script_01_initial_test_output.log
script_02_add_noise_output.log



Scripts should output to both the command line and a log file with a matching name. This keeps all log files in one place, closely associated with their corresponding scripts.

## Purpose

These logs serve as:
1. A record for you to reference later
2. A record for AI—feed these back to recreate context
3. Part of the breadcrumb trail through your experiments
