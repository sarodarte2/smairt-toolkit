# Support

- Usage questions: open a structured **Question** issue.
- Reproducible defects: open a **Bug report** with version, OS, Python, command, and sanitized output.
- Feature proposals: describe the researcher workflow and human or safety gate affected.
- Vulnerabilities: follow [SECURITY.md](SECURITY.md); never report them publicly.

This research preview has no guaranteed support level or response time. Never attach credentials,
raw protected data, private
PDFs, identifiable participant data, or sensitive remote URLs.
