# Changelog

This project is an unreleased research preview. The package version remains `0.2.0-beta.3` for
compatibility, but no GitHub release or tag currently corresponds to that version.

## Unreleased

### Changed

- Reorganized the public documentation around researchers, scientific workflow, architecture,
  safety, harnesses, and development.
- Replaced release-like and verified-demo language with explicit research-preview status.
- Updated repository URLs and public identity for the independent `smairt-toolkit` fork.
- Preserved PNNL and UTEP named color palettes while removing built-in institutional ASCII marks.
- Migrated user-local appearance settings to schema 6 and refreshed generated harness guidance as
  adapter version 8; the shared project schema remains unchanged.

### Added

- Documentation validation for links, structure, terminology, metadata, and public claims.
- A documentation hub, explicit upstream acknowledgments, and clearer GitHub contribution and
  security routes.
- Neutral banner, mark, and social-preview assets plus three focused Mermaid diagrams.

Stable release notes will begin when the project creates its first reviewed GitHub release.
